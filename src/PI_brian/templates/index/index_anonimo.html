Post Me

Este es el proyecto integrado de Brian Holsters para el ciclo formativo de grado superior de Desarrollo de aplicaciones web.

Este proyecto surge de la necesidad de mejorar el sistema que usamos para comunicar nuestras opiniones y emociones con respecto a cualquier tema, aunque ya existan numerosas redes sociales, en este proyecto en concreto se va a explorar la “expansión” que puede suponer asignar a cada mensaje dejado por un usuario de forma obligatoria un “emoticono”, de forma que no exista posibilidad de malinterpretación del mensaje. Cabe destacar que este proyecto no planea sustituir ninguna tecnología ya existente, en lugar de eso tratará de explorar y mostrar un concepto.
